# Tauri + Vue 3

## 项目简介

- 跨平台桌面应用程序
- 使用Vue3、ElementPlus、TailwindCSS
- 使用Tauri、Rust、Vite、Vue3、vue-router

## 项目目录结构

- src/api 接口配置
- src/components 组件
- src/components/layout 布局组件
- src/views 页面
- src/utils 工具
- src/assets 资源
- src/styles 样式
- src/main.js 主程序入口
- src/App.vue 主组件
- src/tauri.conf.json 配置文件
- src/vite.config.js 配置文件
- src/package.json 配置文件
- src/README.md 项目简介
- src-tauri 桌面应用程序
